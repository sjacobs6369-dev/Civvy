# Original mockups

These are the founder's original interactive HTML prototypes — the "original working
product" the React draft in this repo is based on. They are fully self-contained:
every button, tab, and drawer works in plain HTML/CSS/JS.

## How to view them

Just open either file in any browser — no build step, no server:

- Double-click the file, or
- Drag it into a Chrome / Safari / Firefox tab, or
- From a terminal: `open mockups/civvy_mockup_v6.html` (macOS) / `xdg-open` (Linux)

(They load fonts from Google Fonts over the network; everything else is inline, so they
also work offline with a system fallback font.)

## Files

| File | What it shows |
| --- | --- |
| `civvy_mockup_v6.html` | Home feed, Learn (TikTok feed + library + 4-layer reader), My Reps with full profiles and donor transparency, Polis, Passport. The best single-file demo. |
| `civvy_polis_v2.html` | Polis in depth — five sub-tabs including Common Ground (vote-first-then-reveal), polls with context rows, verified constituent polls, Talk Across the Table. |
